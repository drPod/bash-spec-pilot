The implementation uses only the Rust standard library plus libc for Linux renameat2(RENAME_EXCHANGE), which is needed for the documented --exchange option. It covers common GNU mv flags including overwrite policy, target-directory forms, verbose/debug, backups, update modes, and cross-device copy fallback, while intentionally skipping SELinux contexts, ACL/xattr preservation, sparse/reflink behavior, and full metadata fidelity.
